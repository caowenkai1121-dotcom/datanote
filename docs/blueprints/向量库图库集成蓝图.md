# 向量库(Qdrant)+ 图数据库(Neo4j)集成蓝图

> 七角色 Workflow 产出。去零依赖约束后, 实测定为 **REST/HTTP 集成(pom 零新依赖)** 最稳(官方 SDK 多需 Java11/grpc 与 SB2.7 冲突: io.qdrant:client 1.15.0 需排 slf4j; neo4j-java-driver 5.x 需 Java17, 4.4.x 才兼容 Java8 — 均留档备用)。

## 选型
- **向量库 = Qdrant v1.12.4**(Rust 单容器, ~0.5GB 镜像, 几十MB 级内存)。REST 6333, api-key header。集合 dn_meta, 维度=embedding 维度(text-embedding-v3=1024), 换模型=重建集合。
- **图库 = Neo4j 5.26-community**(LTS)。HTTP Query API /db/neo4j/tx/commit, Basic Auth。节点 Table{fqn=db.table 唯一}, 边 FLOWS_TO{jobId,source}。
- **嵌入 = 可配置 OpenAI 兼容 /embeddings**(默认百炼 text-embedding-v3; DeepSeek 无 embeddings 端点)。未配置降级关键字。

## 远端安装(Docker, 38.76.183.50, 密钥经 env 不入提交)
```
docker run -d --name datanote-qdrant --restart unless-stopped -m 1g -p 6333:6333 -v qdrant_storage:/qdrant/storage -e QDRANT__SERVICE__API_KEY=<KEY> qdrant/qdrant:v1.12.4
docker run -d --name datanote-neo4j --restart unless-stopped -m 1200m -p 7474:7474 -p 7687:7687 -e NEO4J_AUTH=neo4j/<PWD> -e NEO4J_server_memory_heap_max__size=512m -e NEO4J_server_memory_pagecache_size=256m -v /opt/neo4j/data:/data -v /opt/neo4j/logs:/logs neo4j:5.26-community
```
本地 app 连远端 env: VECTOR_ENABLED/QDRANT_URL/QDRANT_API_KEY/GRAPH_ENABLED/NEO4J_HTTP/NEO4J_USER/NEO4J_PWD; embedding: EMB_API_KEY 等。

## 降级铁律(已实测)
①工具永返 ok(带 engine: graph|bfs_fallback / vector|keyword_fallback); ②client 短超时+try/catch; ③派生库写绝不进 @Transactional, MySQL 为 SoT; ④@PostConstruct 探活失败静默禁用不阻断启动。

## 里程碑
- **M1 已交付(第58轮)**: 装两库 + GraphStoreClient/GraphMirrorService/VectorStoreClient/EmbeddingService/VectorIndexService/StoreController + 4 工具(graph_impact/trace/neighbors/semantic_search) + 血缘灌图 + 健康端点 + 降级。真 E2E: graphEdges=7, graph_impact engine=graph, semantic_search keyword_fallback。
- **M2**: 配百炼 embedding key → fullReindex 实测向量引擎; 向量索引扩 列/术语/指标(VectorIndexService 增 indexColumns/indexGlossary/indexMetric)。
- **M3**: 增量同步钩子(MetadataCrawlerService.crawlAll/scheduledCrawl 尾部 fullReindex+图镜像, 各自 try/catch); 元数据删除 deleteIds 同步删 point。
- **M4**: 前端血缘图谱直吃 graph_neighbors; 治理血缘视图加 graph_* 情境入口; RAG(向量召回治理文档/术语注入 agent prompt)。
- **M5**: 实体关系图扩展(表↔指标↔规则↔工单↔项目 多类节点/边), 支撑"与 X 相关的一切"查询。

## 工具→真实依据(已核实)
graph_impact→GraphStoreClient Cypher / 回退 LineageEdgeService.impact; graph_trace→/trace; graph_neighbors→/graph; semantic_search→EmbeddingService.embed+VectorStoreClient.search / 回退 DnTableMetaMapper LIKE。
