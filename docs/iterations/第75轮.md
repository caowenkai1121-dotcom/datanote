## 三、设计系统全局固化(业主"应用到全局+未来必用")
- **`docs/设计规范.md` 新增 §10「工程强制规则」**: ①优先用 `DN.*` 组件工厂; ②颜色/间距/圆角/阴影一律走 `var(--*)` 禁硬编码; ③**CSS 变量雷铁律**(局部作用域变量禁被全局规则无 fallback 引用, 共享令牌放 :root, 嵌套 fallback 合法); ④布局防错位铁律(inline-flex select width:auto / 行内徽标 nowrap / 长文本截断 / 网格 align-items:start / 图标放有尺寸容器); ⑤状态齐全(hover/focus/disabled/空态/错误内联); ⑥发布动作(bump ?v 版本 / 改 workspace.html 用编辑器不整文件重写)。
- **项目 `CLAUDE.md` 新增核心原则 6「统一设计系统(强制·全局)」**: 此后所有新增页面/模块/UI 组件及任何 UI 改动必须遵守 `docs/设计规范.md`, 不得另起炉灶; 新做 UI 前先读规范。→ 每个后续 session/agent 自动加载该硬规则。
