## 进度
- 轮次记法: R{n} 模块/主题 — 发现 → 改动 → 验证。

## R1 权限树搜索(系统管理)
- 发现: 角色权限/用户直授弹窗 42 个权限点无搜索, 配权限靠肉眼翻。
- 改: umPermModal+umDirectPerms 加搜索框 + umFilterPermTree(按名称/代码过滤, 组内全隐则隐组)。
- 验: 部署后静态含 umfPermSearch×2/umFilterPermTree×3。

- 发现: 数据源/Doris配置/AI密钥/向量密钥 密码框 type=password 无可视化, 录入易错无法核对。

- 验: 静态含 dnTogglePwd×5。

- 代理报的"删除/批量无确认"经核实=假阳(script/folder/sync-task/bulk 均已 msgConfirm/confirmBox, 全站 confirm 21+处)。确认机制健康。

## R5 正则批量加表 先预览后确认(数据同步)

- 改: 拆为「匹配预览」→ 列出匹配表名+数量 →「确认添加 N 张」两步(dbsyncPreviewBulkAdd/dbsyncConfirmBulkAdd)。
- 验: 静态含 dbsyncConfirmBulkAdd。
