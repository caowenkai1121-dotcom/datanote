# 第76轮 —— 全站右侧抽屉统一为一套设计语言 + 修复 R74 图标回归

> 业主: 系统所有界面包括**右侧画出的页面、抽屉界面都需要统一**。本轮把全站 3 套滑出抽屉收敛为一套视觉语言, 并修掉 R74 遗留的图标撑大回归。

## 一、抽屉乱象盘点
全站并存多套右侧滑出抽屉, 视觉各异:
- `gov-drawer`(DN.drawer): 治理 7 模块 + 主数据 6 模块共 **13 处用**。原 560px/mask .35/shadow -4px/slide .25ease/close 40px。
- `dbsync-drawer`: 数据同步任务详情 + 项目详情。740px/mask .36/shadow -8px/slide .26cubic/close 30px。
- AI `#aiDrawer`(R70 内联): 经验/审批面板。380px/自定义头。
- (内联右栏: `datamap-detail` 数据地图详情, 非滑出, 单独对齐 tab 风格)

## 二、统一为一套抽屉视觉语言(基准 = dbsync-drawer, 全令牌化)
| 维度 | 统一值 |
|---|---|
| 遮罩 | `var(--bg-mask)` rgba(15,20,40,.45) |
| 阴影 | `-8px 0 32px rgba(15,20,40,.14)` |
| 滑入 | `transform .26s cubic-bezier(.4,0,.2,1)` |
| 宽度 | 标准 **600px**(gov-drawer)/ 宽 **740px**(dbsync 多tab详情), max 94vw |
| 头部 | `16px 20px` + bg-card + divider 底边, 标题 `16px/600` 省略号 |
| 关闭 | `32×32` + `--radius` + muted→hover bg-hover |
| 主体 | `18px 20px` flex 滚动 |

- **gov-drawer**(gov-modern.css): 560→600px, 全部 chrome 对齐 + 令牌化, 标题加省略号(13 模块一次统一)。
- **dbsync-drawer**(app.css): mask/阴影令牌化, close 30→32px + radius 统一, head padding 15→16。
- **AI #aiDrawer**(ai-agent.js): 头 16/20 + 标题 16/600 + 关闭 32×32 + 阴影 -8px, 与抽屉同族。
- **datamap-detail tab**(app.css): 去左侧 3px 竖条 + 硬编码蓝底, 改底部下划线(与 drawer tab 同款), 返回按钮 hover 改 `--primary-light`。

## 三、顺手修 R74 图标撑大回归(业主"修复所有问题UI")
- **根因**: R74 给 `DN.icon` 加 `width/height="100%"` 修 AI 巨型图标; 但 SVG 在**无尺寸 span** 里 100% 会撑成 300×300 巨图(治理待办等处出现金色巨标)。
- **正解**: `DN.icon` 默认改 `1em`(随容器 font-size, 永不撑大), CSS `svg{width:Npx}` 规则优先级更高仍覆盖到精确尺寸(header16/stat22/empty44 等不变); AI 欢迎/顶栏图标改用容器 `font-size` 控大小(40/26px)。
- 实证: 治理页**非图表超大 svg 由 2 个→0**; AI 欢迎图标 40×40、顶栏 26×26 恢复。

## 四、统一标准入文档(未来抽屉必用)
