# 第 28 轮 · 全局深链返回栈/面包屑 + 数据地图→数据集闭环(大版本)

> R21..R44 第八轮。深链下钻遍布全系统后, 用户易"迷路"。本轮加全局返回栈 + 浮动「返回来源」chip, 并补数据地图→数据集闭环。

## 改动(workspace.html)
- **深链返回栈**: `navigateTo(route,ctx)` 带 ctx(深链下钻)时把当前来源压 `__navStack`; 普通点顶部 Tab(无 ctx)清空栈。`window.navBack()` 弹栈返回上一来源。
- **浮动「返回来源」chip**: 左下角固定按钮, 栈非空时显示"← 返回<来源模块>（层数）", 点击 navBack, 栈空自动隐藏。深链后不再迷路。
- **数据地图→数据集闭环**: 表详情 DDL/SQL Tab 的 SELECT 块加「存为数据集」→ `navigateTo('governance',{gov:'consumption',newDataset:{db,table}})`, 消费层(R21)自动开新建数据集抽屉并预填 SQL。

## QA —— ✅ 通过(Playwright)
33 UI 绿; 本地运行。Playwright 实测: 深链 catalog→governance/quality 压栈=1, chip 显示「← 返回数据地图」; `navBack()` 回到 catalog, 栈清空, chip 隐藏; `dmTableToDataset` 就位。
