## 一、/goal 有界自驱(W3)
- **AiAgentService.pursue(sessionId, goal, ctx, maxCycles)**: 围绕目标连续推进多周期(每周期一次完整 run), 直到 ①agent 答复含『目标已达成』②非 done(挂起/阻塞/中断)③本周期无工具进展(停滞)④达周期上限。每周期受 run 的 budget+墙钟约束 → **多重有界, 无失控 auto-loop**。
- **Controller POST /pursue**(body {message, sessionId?, maxCycles?}, 默认2/上限3)。
- E2E: "盘点治理现状并给改进建议" maxCycles=2 → status=done **cycles=2** 188s, 答复完整。
