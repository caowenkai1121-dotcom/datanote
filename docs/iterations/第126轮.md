# 第126轮 — MDM↔数据标准打通(属性绑定数据元+词根校验) + 黄金记录变更历史/版本diff + 色值令牌化

【Domain】主数据建模与数据标准两模块此前互为孤岛:属性命名/类型随手填,无落标依据;黄金记录只有 version 数字,改了什么无从追溯。本轮打通两条联动:①属性落标(绑定数据元+词根命名校验)②黄金记录全变更链可回溯可 diff。

## 各角色产出

【Architect】
- `dn_mdm_attribute` 加 `element_code`(关联 `dn_data_element.element_code`,落标即元数据级联动,不复制数据元内容,避免双写);

【Product】
- 属性表单新增「绑定数据元」下拉(数据标准·数据元),选中自动带入名称/类型/长度建议(标准类型→MDM类型映射 mapElemType);已绑定但数据元被删时下拉显示「已不存在」不丢值;
