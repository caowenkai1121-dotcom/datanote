# 第 24 轮 · 系统监控/运维统计可跳处理(消除监控死胡同·大版本)

> R21..R44 第四轮。架构师:系统监控大屏、运维统计是"看得见摸不着"的只读死胡同——告警/离线/统计不能跳处理。本轮全部接通处理动作。

## 改动(workspace.html)
- **数据集成统计**: 每行 `syncTask:id` → `opsGotoSyncTask(id)` → `navigateTo('dbsync',{openDetail:id})` 直达该同步任务详情抽屉。
- **dbsync 路由 init**: 消费 `__navCtx.openDetail` → 自动打开任务详情(深链接收端)。
- **系统监控大屏**: 调度统计块 → 点击进入数据运维; 审计预览 → 「查看全部审计」跳治理审计中心; 服务离线行 → 点击跳数据源/环境配置排查。
- **系统管理路由 init**: 消费 `__navCtx.sm` → 自动切到对应子 Tab(datasource/monitor/...), 使外部可深链直达系统管理任一子页。

## 业务闭环(本轮)
变更影响/运维闭环: 集成统计→同步任务详情; 监控告警→运维/数据源/审计 直达处理, 监控不再只读。
