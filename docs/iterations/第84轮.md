→ **所有抽屉/弹窗表单输入(含尚未逐个改造的 gov/mdm-* 表单)即刻变舒适统一, 不触 JS。**

## 二、旗舰改造: gov-quality「新建质量规则」
从扁平 `formRow` 堆叠 → 新系统:
- 3 分组: 规则定义 / 检查目标 / 阈值与级别(靛蓝竖条标题)。
- 必填红星: 规则名称/数据库/表名。
- 两列(grid2): 类型+数据源 / 库+表 / 字段+维度 / 阈值+级别 —— 均衡不冗长。
- 人性化提示: "简明描述本规则检查什么" / "自定义SQL类型可空" / "低于此值判为不通过"。
- sticky 底栏: 取消 + 创建规则(DN.drawerFoot, busy/reset 态)。
