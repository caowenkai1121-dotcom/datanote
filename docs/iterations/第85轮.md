# 第85轮 —— 全站抽屉表单批量改造(7 路并行, ~18 表单)

## 一、并行改造(7 个 general-purpose subagent, 各 1~3 文件, 互不冲突)
统一改造配方(沿用现有 input 元素与全部业务/校验/请求逻辑零改, 只换呈现+按钮收尾): 字段按语义分 `DN.formSection` + `DN.field({required,hint})` + 短字段 `DN.formGrid2` 两列; body 内保存按钮 → `DN.drawerFoot({okText,onOk,onCancel})` + `DN.drawer(title,body,foot.el)` sticky 贴底; busy 态走 `foot.busy()/reset()`。
