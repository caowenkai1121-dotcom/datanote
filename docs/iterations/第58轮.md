## 二、Java 集成(REST/HTTP, 零新依赖, 降级优先)
| 组件 | 说明 |
|---|---|
| `GraphStoreClient` | Neo4j HTTP Query API(/db/neo4j/tx/commit, Basic Auth, HttpURLConnection 连3s读10s)。@PostConstruct 探活+建唯一约束(Table.fqn), available() 守门, 失败静默禁用不阻断启动。run(cypher,params 参数化防注入)。 |
| `GraphMirrorService` | 血缘 MySQL→图库镜像。fullSync(): 先删 source=MAPPING 边, UNWIND 批量 MERGE 节点+FLOWS_TO 边(幂等)。try/catch 隔离, **绝不回滚 MySQL(SoT)**。 |
| `VectorStoreClient` | Qdrant REST(api-key header, 连5s读15s)。探活/ensureCollection(dim 幂等)/upsert/search(向量+kind过滤)。 |
| `EmbeddingService` | OpenAI 兼容 /embeddings(默认百炼 text-embedding-v3 1024维, DB配置优先+env兜底+CryptoUtil解密)。未配置 isAvailable()=false → 降级。 |
| `VectorIndexService` | 元数据→向量(本轮表元数据, 列/术语/指标顺延)。向量/embedding 不可用直接 return。point id=UUID(kind:id) 稳定更新。 |
| `StoreController` | GET /api/ai/store/health(三态)+ POST /api/ai/store/sync(手动灌库)。 |
| `LineageEdgeService`(改) | rebuildFromSyncJobs() 尾部挂钩 graphMirrorService.fullSync() try/catch, 图写失败不影响主流程。 |

## 三、AI Agent 新工具(4, 图优先/向量优先 + 自动回退, 永返 ok 带 engine 标记)
- `graph_impact`(图库多跳下游, Cypher FLOWS_TO*; 回退 LineageEdgeService.impact Java BFS)
- `graph_trace`(图库多跳上游溯源; 回退 trace)
- `graph_neighbors`(N跳双向子图; 回退 graph())
- `semantic_search`(向量库+embedding 语义检索表/列/术语/指标; 回退关键字 LIKE)
- 工具数 **9→13**。配置键: application.yml 加 `datanote.ai.embedding-*` + `datanote.vector.*` + `datanote.graph.*`(全 ${ENV:默认}, 默认 enabled=false)。
