# 第127轮 — 黄金记录发布扇出异步化 + 全站 color:#hex 令牌清扫(21处/10文件)

## 本轮目标
【Domain】R126 遗留:发布黄金记录要等 10s+ 才返回,真实生产不可接受(用户以为卡死会狂点)。另设计规范 §10 禁硬编码 #hex,存量 JS 仍有 37 处 color:#hex。

## 各角色产出

【Architect】根因:MdmPublishService.fanOut 对每个订阅做同步 HTTP 推送(连接/读各 3s 超时),死端点订阅 N 个 = N×6s 拖死发布响应。定案:**黄金记录发布的自动扇出改后台异步**(CompletableFuture.runAsync,推送结果照落 dn_mdm_publish_log 可查不丢审计);治理中心手动发布(MdmGovernanceController 用 fanOut 返回值显示命中数)保持同步——用户主动等结果,语义不同不动。

【Dev-A】MdmGoldenController.publish:fanOut 调用包进 CompletableFuture.runAsync(原本就 try-catch 忽略结果,异步化无语义损失)。

【Design+Dev-B】color:#hex 令牌化(只动裸 hex,不碰 var(--x,#hex) fallback 既有约定):
- `#e03131/#f53f3f → var(--error)`、`#389e0d/#00b42a → var(--success)`、`#ff7d00/#d4860a → var(--warning)`、`#3457d5 → var(--primary,#3457d5)`、`#86909c/#5b6472 → var(--text-muted)`;
- 共 21 处替换,涉及 gov-quality/mdm-hierarchy/mdm-pubsub/mdm-survivorship/mdm-refdata/mdm-approval/gov-assets/gov-audit/dn-common/ai-agent 共 10 文件,版本号全部 bump r127;
- **有意保留**:`color:#fff`(彩色底反白字,无对应令牌)、`#9a7b1a/#9a5b00`(浅色底上有意加深的琥珀,直换 --warning 会降对比度)、`#722ed1`(紫,无令牌)——留待设计逐个定案。
