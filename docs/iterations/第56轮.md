- **engine 层**：`AiAgentService`(单步/多步顺序循环：注入工具清单→模型回 `<tool_call>`→Java 执行回灌→直到终答或达 MAX_STEPS=6) / `AgentTextUtil`(sanitize 剥NUL/控制字符、parseToolCalls、extractJson 平衡大括号容错) / `PromptBuilder`(三层拼装) / `Validation`(粗校验) / `AgentState`。
- **web**：`AiAgentController` /api/ai/agent/{chat, session/{id}, tools}，全返 R<T>。
- **前端**：workspace.html 加 `assistant` 顶级路由 + `viewAssistant` 容器 + header-tab；新增 `js/ai-agent.js`(纯 DN.* 组件对话面板，工具卡逐道展示，一问一闭环)。
