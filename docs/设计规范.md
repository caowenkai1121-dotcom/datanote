# DataNote 设计系统 v3 —「墨青 Ink & Indigo」

> 本文件是全站**唯一**设计契约。所有新增页面、模块、组件与任何 UI 改动必须遵守。
> 实现载体:`css/dn-design.css`(令牌+组件+遗留适配,最后加载) + `js/dn-common.js`(DN.* 组件工厂)。
> v3 取代 v2「Slate & Indigo」;v2 的 §10 工程铁律全部保留并在 §10 更新。

---

## 0. 设计立场

数据密集型中文企业平台。立场四条:

1. **静默专业**:界面退后,数据向前。色彩克制(中性占 95%,语义色只在状态处出现),无装饰性渐变、无彩条、无 emoji 图标。
2. **单一来源**:一种主色、一族灰、一套表单、一套表格、一套弹窗、一套抽屉、一套图标(20px 1.6 描边 SVG)。同类问题禁止第二套实现。
3. **密度优先,呼吸为界**:数据区紧凑(13px 表格),区块间留白慷慨(24/32px),用空间而非线框分隔。
4. **状态齐全**:每个交互组件必须有 hover / active / focus-visible / disabled;每个数据区必须有 loading(骨架) / empty / error(带重试) 三态。

## 3. 间距 / 圆角 / 阴影 / 层级 / 动效

### 3.1 间距(4px 基,禁止其它 margin/padding 值)
`--sp-1: 4px` `--sp-2: 8px` `--sp-3: 12px` `--sp-4: 16px` `--sp-5: 20px` `--sp-6: 24px` `--sp-8: 32px` `--sp-10: 40px`
- 控件内部:纵 `--sp-2` 横 `--sp-3~4`;卡片内边距 `--sp-4~5`;卡片间距 `--sp-4`;区块间距 `--sp-6~8`。

### 3.2 圆角(内紧外松)
`--radius-xs: 4px`(徽标/复选) `--radius-sm: 6px`(按钮/输入) `--radius-md: 8px`(下拉/小卡) `--radius-lg: 12px`(卡片) `--radius-xl: 16px`(弹窗/hero) `--radius-full: 999px`(药丸)

### 3.3 阴影(靛染,统一上方光源;禁纯黑阴影)
| 令牌 | 用途 |
|---|---|
| `--shadow-sm` | 控件浮起 `0 1px 2px rgba(23,30,60,.05)` |
| `--shadow-md` | 卡片 `0 1px 2px rgba(23,30,60,.04), 0 6px 20px rgba(23,30,60,.06)` |
| `--shadow-lg` | 弹窗/抽屉 `0 16px 48px rgba(23,30,60,.18)` |
| `--shadow-hover` | 卡片悬浮 `0 8px 24px rgba(64,81,211,.13)` |

### 3.4 z-index(禁止字面量 z-index)
`--z-sticky: 100`(吸顶表头) `--z-panel: 500`(侧栏浮层) `--z-dropdown: 700` `--z-drawer: 800` `--z-modal: 900` `--z-toast: 1000` `--z-spotlight: 1100`(Ctrl+K/帮助)
- 同层内用 +1(遮罩 800、抽屉 801)。

### 3.5 动效
`--dur-fast: .12s`(按压/着色) `--dur: .2s`(hover/淡入) `--dur-slow: .3s`(抽屉/弹窗进出场) `--ease: cubic-bezier(.4,0,.2,1)` `--ease-out: cubic-bezier(0,0,.2,1)`
- 只动 `transform`/`opacity`(GPU);禁 top/left/width/height 动画。
- 进场:淡入+上移 4px;列表可错峰(每项 +30ms,≤8 项)。
- 弹窗:scale(.98)→1 + 淡入;抽屉:translateX(100%)→0。
- `prefers-reduced-motion: reduce` 时全部动效退化为即时。

## 4. 组件规范(全部经 DN.* 工厂或 dn-design.css 类,禁手拼)

### 4.1 按钮 `.btn`
- 高 32px(`.btn-sm` 28px / `.btn-lg` 38px),radius-sm,fs-sm,字重 500,inline-flex + gap 6px。
- 变体:`btn-primary`(实心主色,**禁渐变**) / `btn`(默认:白底描边) / `btn-danger`(error-bg 底 error-text 字) / `btn-ghost`(透明,主色字) / 文字链接仅用于表格行内操作。
- 状态:hover 深一档背景;active `translateY(1px)`;focus-visible 2px 主色外环(offset 2px);disabled 50% 透明 + `pointer-events:none`(走 `:disabled` 或 `.is-disabled`,禁 JS 改 style)。
- 加载中:`.is-loading` 自带 spinner,文字保留,禁手改 textContent+opacity。
- 行内操作链接:类 `.dn-link`(主色)/`.dn-link.danger`(错色),禁裸 `<a style="color:...">`。

### 4.2 卡片 `.gov-card`(DN.card)
- 白底、border 1px、radius-lg、shadow-md;hover(可点卡)`translateY(-2px)` + shadow-hover。
- **禁左侧彩条**(v2 ::before 彩条在 v3 移除);语义 tone 由图标芯片色承载。
- 头部:fs-md 600,图标 18px 主色;体:padding `--sp-4~5`。

### 4.3 KPI 磁贴 `.gov-stat`(DN.statRow)
- 数字 fs-3xl 700 tabular;标签 fs-xs muted;图标芯片 40px radius-md,tone 决定芯片色(ok/warn/err/info)。
- 可点磁贴:cursor + hover 浮起 + tabindex(已内置)。

### 4.4 表格 `.gov-tbl`(DN.table)
- 表头:sunken 底、fs-xs 600 muted、letter-spacing .04em、sticky;行:fs-sm,行高≥44px,divider 分隔,hover primary-light(无左标记条);斑马纹移除(干扰扫读)。
- 数值列右对齐 + tabular;操作列用 `.dn-link`,≤3 个,更多收进「⋯」。
- 必带:搜索(可关)、分页、空态、加载骨架。

### 4.5 表单(DN.field / formSection / formInput / formSelect)
- **唯一表单体系 `dn-form-*`**:控件高 34px,radius-sm,border,focus 主色边+3px ring;`iw-form-select` 等旧类由适配层重皮,新代码禁用旧类。
- 标签上置 fs-sm 500,必填红星;提示 `dn-hint` fs-xs muted;错误内联 `dn-error` fs-xs error-text(禁 alert)。
- 分组 `dn-form-section` + `dn-sec-title`;短字段 `dn-form-grid2` 两列。
- textarea 最小高 72px;禁把 textarea 塞进两列半栏。

### 4.6 抽屉(DN.drawer)
- 600px(宽内容 720px)/max 94vw,shadow-lg,slide-in --dur-slow;头 fs-lg 600 + 关闭钮;底部操作栏 sticky(`drawerFoot`:取消(默认)+主按钮)。
- Esc 关闭、焦点陷阱、归还焦点(已内置)。表单一律抽屉,**禁新增弹窗表单**。

### 4.7 弹窗(确认/告知)
- 唯一类 `dn-modal`(遮罩 --z-modal):radius-xl,宽 420px,标题 fs-lg,操作右对齐。
- **`window.confirm/alert/prompt` 全站禁用**,破坏性确认用 `DN.confirm()`(危险按钮红色,文案具体到对象名)。

### 4.8 徽标/药丸 `.gov-pill`
- fs-xs 600,padding 2px 8px,radius-full,语义四件套配色(bg+text+ring 内描边)。
- 状态点:`.dn-dot`(8px 圆点+文字)用于运行态。

### 4.9 反馈
- Toast(`dn-toast`):右上角,fs-sm,白底+左侧语义条 3px+shadow-lg,z=--z-toast,2.6s;成功不带感叹号。
- 空态(DN.empty):图标 44px 25% 透明 + 一句话 + 可选主操作按钮;文案具体(「暂无同步任务,点右上角新建」),禁 emoji。
- 错误态(DN.errorBox):alert 图标 + 原因 + 重试按钮。
- 骨架(DN.skeleton):shimmer,与目标布局同形。

### 4.10 图标
- 唯一体系:DN.icon 20×20 viewBox、stroke 1.6、round cap。新增图标进 ICONS 表,**禁 emoji、禁第二套图标库**。

## 5. 布局与外壳

- 顶栏 52px:表面色+底边框(禁纯白漂浮);导航项 active 为 2px 主色下划线 + 主色字;溢出滚动隐藏滚动条。
- 内容区 max-width 1600px,页面内边距 `--sp-6` 起。
- 页面结构:H1(fs-2xl)+ 一句话描述(muted)→ KPI 行 → 内容卡片;模块侧栏 200px。
- 栅格:`dash-grid`(12 列)/`gov-grid`(auto-fill 260px);断点统一 **1200 / 768 / 520** 三档,禁其它断点值。
- 滚动条:全站统一细滚动条(8px,thumb 用 border 色,hover 加深)。

## 6. 文案

- 成功「已保存」「已删除」——不带感叹号;错误直说原因+下一步「连接失败,请检查数据源配置」;禁「哎呀/Oops」。
- 按钮动词在前:「新建任务」「导出 CSV」;全站「+ 新建」用半角 +。
- 数字带千分位;时间相对(悬停见绝对);空值统一「-」。

## 7. 图表

- 单序列图一律主色;多序列用图表专用板 `--chart-1..6`(dn-design.css 定义:靛蓝/青/琥珀/玫红/灰蓝/苔绿,饱和度≤70%),禁随手取色。
- 仪表/进度按值着色只用语义三色;轴线/网格用 `--divider`,标签 `--text-faint`。

## 10. 工程铁律(强制,QA 按此打回)

1. **10.1 颜色/尺寸全令牌**:禁 #hex、rgb() 字面量、魔法 px(间距/字号/圆角/阴影/z-index 全走 §1-§3 令牌)。`var(--x,#hex)` 带 hex 兜底=违规。
2. **10.2 组件唯一**:同类 UI 必须用 DN.* 工厂/规范类;发现第二套实现(自拼 modal、手搓表格、自注入 style)按缺陷处理,先沉淀进 dn-common/dn-design 再使用。
3. **10.3 CSS 变量雷**:局部作用域定义的变量禁止被全局规则无 fallback 引用(历史教训:全站隐形按钮)。v3 下所有令牌定义在 `:root`,新增局部令牌须评审。
4. **10.4 布局防错位**:inline-flex 选择器需 width:auto;行内徽标 nowrap;长文本必截断(ellipsis+title);网格 align-items:start;表格容器 overflow-x:auto。
5. **10.5 状态齐全**:交互件 hover/active/focus-visible/disabled 四态,数据区 loading/empty/error 三态,表单错误内联展示。禁 JS 改 style 模拟状态(用类切换)。
6. **10.6 弹层纪律**:z-index 只用 §3.4 令牌;表单进抽屉;确认用 DN.confirm;`window.confirm/alert/prompt` 禁用。
7. **10.7 可达性**:弹层 role=dialog+aria-modal+焦点陷阱+Esc;可点非按钮元素加 role=button+tabindex+键盘处理;图标按钮带 aria-label。
8. **10.8 静态资源**:CSS 只在 `<head>` 引;改 js/css 必 bump `?v=`(v3 起用 `u<N>` 序列);新页面必须先读本文件。
