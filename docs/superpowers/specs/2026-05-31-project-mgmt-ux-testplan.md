# 项目管理模块 — 全面 UX 测试案例矩阵

测试环境：http://38.76.183.50:8099/workspace.html?v=N#/project（带 ?v 破缓存）。鉴权开放(anonymous→admin)。
原则：用户体验优先，全覆盖；功能/边界/状态机/交互/空态/校验/层级/一致性。
记号：✅通过 ⚠️UX问题 ❌缺陷

## A. 项目空间 — 列表/筛选/CRUD
- A1 列表加载+KPI(总数/活跃/归档/本月新增)+计数"共N/M"
- A2 新建项目：必填校验(空名拦截) / 自动编码(中文名→PROJ_id, 英文→slug) / 默认值
- A3 新建项目：模板预填(数据同步/SQL开发/数仓混合)切换填充类型环境标签
- A4 编辑项目：回填字段 / 改名改类型 / 保存生效 / 编码冲突追加_2
- A5 搜索：按名称/编码/负责人模糊；清空恢复
- A6 筛选：类型/状态/环境/标签 单选与组合；计数同步
- A7 排序：最新/最早/名称/收藏优先/最近访问
- A8 仅收藏复选框
- A9 收藏星切换(☆↔★) + 工作台/筛选联动
- A10 置顶/取消置顶 + 排序"收藏优先"体现
- A11 归档：confirm→ARCHIVED→状态筛选可见；归档后行内"归档"消失
- A12 删除：confirm→消失→KPI减1；已删不再出现
- A13 标签管理弹窗：建标签(空名拦截/重名拦截)/删标签(confirm,移除关联)/颜色
- A14 行内"标签"弹窗：勾选保存→列表彩色chip；legacy tags 灰chip并存
- A15 空态文案(无项目/无匹配)

## B. 详情抽屉 — 概览
- B1 打开抽屉:头部(名/状态badge/编码·类型·环境·负责人) / 8页签 / 关闭(×/mask)
- B2 健康分卡:总分+等级+5维进度条+/20标识;空项目分值合理
- B3 概览KPI(资产/成员/发布/待审批) 与实际一致
- B4 资产分布(4类计数)
- B5 绑定任务运行聚合(成功/失败/运行中/未运行) — 有绑定同步任务时
- B6 最近活动流(成员加入等) 时间/操作人
- B7 公告卡:发布公告(空标题拦截/优先级)/删除(confirm)/[重要]标记/空态

## C. 详情 — 资产
- C1 类型切换(同步任务/脚本/数据源/质量规则)→候选刷新
- C2 绑定资产→表格出现+候选移除已绑
- C3 重复绑定拦截(同资产)
- C4 归属反查→toast显示所属项目
- C5 解绑(confirm)→消失+候选恢复
- C6 空态

## D. 详情 — 成员 + 邀请
- D1 成员列表(admin/负责人)
- D2 加成员(空名拦截/重复拦截/非法角色)→出现
- D3 改角色下拉(无confirm,即时生效)
- D4 最后OWNER护栏:不可降级/移除最后负责人(报错toast)
- D5 移除成员(confirm)
- D6 从用户列表选(prompt)
- D7 建邀请(指定/凭码)→PENDING行
- D8 邀请码复制(完整token)
- D9 接受邀请→ACCEPTED+加成员
- D10 取消邀请→CANCELED 【UX:无confirm 待评估】
- D11 空态(无成员/无邀请)

## E. 详情 — 任务 + 里程碑
- E1 计数(待办/进行中/完成)
- E2 新建任务(空标题拦截/优先级/状态/指派/截止/里程碑)
- E3 任务按状态分组展示+优先级颜色
- E4 状态下拉即时切换→重新分组+计数更新
- E5 编辑任务回填
- E6 删除任务(confirm)
- E7 里程碑管理:建(空名拦截)/列表/删(confirm)
- E8 任务挂里程碑→删里程碑后任务解除关联(不报错)
- E9 空态

## F. 详情 — 发布(状态机)
- F1 提交发布(默认标题/环境/内容)→v1 PENDING
- F2 版本号自增 v1→v2
- F3 通过(prompt意见)→APPROVED
- F4 驳回(prompt原因)→REJECTED(终态无操作)
- F5 上线(confirm)→RELEASED
- F6 回滚(confirm)→ROLLED_BACK(终态)
- F7 非法转移防护(终态无操作链接)
- F8 从抽屉内打开发布弹窗 提交按钮可点(z-index回归)
- F9 空态

## G. 详情 — 文档Wiki
- G1 新建页面(空标题拦截)→树出现
- G2 Markdown渲染(H1-6/列表/粗斜体/代码块/引用/链接)
- G3 XSS净化(script转义/javascript:剥离)
- G4 子页面(选父级)→树两级缩进
- G5 选中高亮+阅读区(更新人/时间)
- G6 编辑回填
- G7 删除(confirm,子页提升为根)
- G8 列表不含content大字段
- G9 空态

## H. 详情 — 活动 + 设置
- H1 活动日志(时间/操作人/方法/路径/状态;>=400红)
- H2 设置-基本信息展示(含标签/描述)
- H3 设置-编辑/存为模板(prompt)/归档/删除 按钮
- H4 配额表单:读默认/改保存/重载持久
- H5 环境参数:新增(空键拦截)/覆盖更新/删除(confirm,归属校验)/空态

## I. 工作台 + 模板 + 对比 + 成员权限 + 发布管理
- I1 工作台5卡(我负责/收藏/最近/待审批/我的待办);已删项目不出现孤儿
- I2 工作台卡片点击跳详情/去处理
- I3 模板管理:列表/从模板新建(空名拦截)/删除(confirm)/空态
- I4 存为模板→模板管理出现→从模板建项目(带成员/类型/环境)
- I5 多项目对比:<2拦截/选≥2→对比表(资产/成员/发布/待审批)
- I6 成员权限矩阵(5角色×权限点)
- I7 发布管理跨项目中心:状态筛选芯片/操作同发布页签/空态

## J. UX 横切 / 边界 / 一致性
- J1 极长文本(项目名/标签/Wiki标题)不破版
- J2 特殊字符/XSS(名称/描述/公告/任务标题) 转义安全
- J3 模态框Esc关闭? 背景点击关闭? 焦点
- J4 加载态"加载中"/失败态展示
- J5 窄屏(<768)布局可用性
- J6 抽屉内开弹窗层级正确(z-index)
- J7 计数与展示一致(KPI/筛选/任务分组/工作台)
- J8 操作后即时刷新(增删改后列表/抽屉更新)
- J9 截断可读性(编码/邀请码/资产名 复制或tooltip)
- J10 健康分数据异常防护(run≤20)
